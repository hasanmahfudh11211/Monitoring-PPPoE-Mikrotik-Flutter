Pertanyaan Umum (FAQ)

Jawaban atas pertanyaan yang sering diajukan.

Apakah aplikasi ini gratis?

Ya, aplikasi Mikrotik Monitor dapat diunduh dan digunakan secara gratis dengan fitur dasar yang lengkap.

Router Mikrotik apa saja yang didukung?

Aplikasi ini mendukung hampir semua jenis router Mikrotik yang menjalankan RouterOS v6 maupun v7 dengan API service yang diaktifkan.

Bagaimana cara mengaktifkan API di Mikrotik?

Anda bisa mengaktifkannya melalui Winbox di menu IP > Services, lalu aktifkan service 'api' (port default 8728).
