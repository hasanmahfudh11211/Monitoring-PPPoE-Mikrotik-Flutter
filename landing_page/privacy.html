Kebijakan Privasi

Terakhir diperbarui: 21 Desember 2025

1. Pendahuluan

Mikrotik Monitor ("kami") menghargai privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda saat menggunakan aplikasi mobile kami.

2. Informasi yang Kami Kumpulkan

Aplikasi ini tidak mengumpulkan data pribadi pengguna secara eksternal. Semua data kredensial router (IP, Username, Password) hanya disimpan secara lokal di perangkat Anda (encrypted) dan digunakan semata-mata untuk menghubungkan aplikasi ke router Mikrotik Anda melalui API.

3. Penggunaan Izin (Permissions)

Aplikasi mungkin meminta izin berikut:

  • Internet: Untuk menghubungkan aplikasi ke router Mikrotik Anda.
  • Penyimpanan: Untuk menyimpan log atau backup konfigurasi jika diminta oleh pengguna.

4. Keamanan Data

Kami menerapkan langkah-langkah keamanan standar industri untuk melindungi kredensial yang tersimpan di perangkat Anda. Kami tidak pernah mengirimkan kredensial router Anda ke server pihak ketiga mana pun.

5. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan diberitahukan melalui pembaruan aplikasi atau halaman ini.

6. Hubungi Kami

Jika Anda memiliki pertanyaan tentang Kebijakan Privasi ini, silakan hubungi kami melalui halaman Kontak.
